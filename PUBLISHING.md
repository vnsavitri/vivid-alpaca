# Publishing Boundary

This repository is intentionally a limited public showcase.

## Public

- High-level README
- Architecture and UI screenshots
- Upstream attribution
- Safety posture
- License

## Private

- Full runnable source code
- API integrations and execution implementation
- Detailed prompts and guardrail implementation
- Session notes and project memory
- Account screenshots, balances, positions, and trade history
- Trade playbooks and proprietary learning materials

The goal is to demonstrate the product direction and engineering judgment without releasing a turnkey system or private training process.
